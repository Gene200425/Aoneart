# Images pour le site A-OneArt

## 📸 Images requises

### Image de fond pour la page d'accueil (hero-bg.jpg)
- **Dimensions recommandées**: 1920x1080px (ou plus grand pour la qualité)
- **Format**: JPG ou PNG
- **Contenu suggéré**: 
  - Matériel d'artiste (pinceaux, peintures, toiles)
  - Atelier d'artiste
  - Œuvres d'art abstraites
  - Nature morte artistique
- **Style**: Image claire et artistique, pas trop chargée pour ne pas distraire du texte

### Images du portfolio (portfolio1.jpg à portfolio6.jpg)
- **Dimensions**: 800x800px minimum
- **Format**: JPG de préférence
- **Contenu**: Vos créations en sérigraphie, portraits et dessins

### Images de l'équipe (team1.jpg à team3.jpg)
- **Dimensions**: 400x400px
- **Format**: JPG
- **Contenu**: Photos professionnelles de l'équipe

## 🎨 Conseils pour les images

1. **Qualité**: Utilisez des images haute résolution (au moins 72 DPI)
2. **Taille**: Optimisez pour le web (compresser sans perdre qualité)
3. **Droits**: Assurez-vous d'avoir les droits d'utilisation des images
4. **Cohérence**: Maintenez un style visuel cohérent

## 📝 Comment remplacer les images

1. Placez vos images dans ce dossier `images/`
2. Nommez-les exactement comme indiqué ci-dessus
3. Le site se mettra à jour automatiquement
